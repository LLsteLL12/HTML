Diacritical Marks

A diacritical mark is used on characters to show accents for example.

Guard in Polish would be Stranz̀ik.

Using the examplary word "guard" you would add the accent by coding "Stranz& #768 ;ik". Showing the code needed to be after the accented character.

̀ would be "(character) & #768; "

́ would be "(character) & #769;"

̂ would be "(character) & #770;"

̃ would be "(character) & #771;"
