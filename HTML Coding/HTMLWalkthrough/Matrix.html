If we wanted to rotate a image on a 3d axis like so

We would use the code :-
.transformed {
-webkit-transform:rotate3d(0, 0, 0, 0deg); /* Chrome, Safari, */
transform:rotate3d(0, 0, 0, 0deg); /* Standard syntax */
transition:transform 3s;
}

For all 3d transformations we need 3 values, X,Y,Z in that order. The rotate needs the "angle rotate3d(X, Y, Z, A)", scale/translate (X, Y, Z)

.transformed:hover{
transform:rotate3d(-2, -1, -2, -360deg);
}
This is the Standard syntax.

We can also alter the sizing editing the ".transformed:hover {"transform:scale3d(1, 1, -1);"}" too ".transformed:hover {"transform:scale3d(3, 2, -3);"}"

That would look like this.

Notice how the use of negative "Z" value has made the image rotate. If it was a positive value the image would simply "zoom in" to enlarge.

Using "transform:translate3d(42px, -62px, -1350px);" however, simply manipulates the image too move in a direction given using pixels. The image below will move North East as you hover over it. switch the negative values for positives however and the image will move South West.
