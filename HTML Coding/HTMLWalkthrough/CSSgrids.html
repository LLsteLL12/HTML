CSS Grid Intro

CSS grids are made up of a series of values equating too 100% and can look like the following format:-


< div class="grid-container">
< header class="item1">

< /header>

< div class="item2" id="menu">
< /div>

< div class="item3" id="main">
< /div>

< div class="item4" id="right">
< /div>

< footer class="item5">

< /footer>

< /div>3

The first "div" container will be treated as the "parent" and those beneath are to be considered "children". notice the parent has its own CSS class alongside each child.
