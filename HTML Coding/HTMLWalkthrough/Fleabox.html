Flex Box

Flexboxes are used to contain a number of items as almost like a command list. It is used as a parent container and will command child elements and tags what to do.

There are a number of different attributes e could use alongside "flex" see Flexbox Game or walkthrough for an interactive learning game or a walkthrough of how to use Flexboxes.


An example of this would look like :-
< div class="parent-container">
< div class="child-item">
< p> Item 1< /p>
< /div>
< div class="child-item">

< /div>
