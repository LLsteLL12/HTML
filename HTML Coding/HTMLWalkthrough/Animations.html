Creating Animations and Keyframes.

An animation brings life to our page to life by giving gradual change too an element from one style to another. We can change as many CSS properties you want, as frquently as we woukd like.

To use CSS animation, we must first specify some keyframes for the animation using CSS. Keyframes hold what styles the elements will have at certain times and for durations of our choosing.



We would code it using CSS as follows :-
@keyframes simple {
from{
background-color: blue; }
to { background-color:purple; }
} button:hover {
color: white;
background-color: blue;

animation-name: simple;
animation-duration: 4s; }

The above code would give us the affect on the button below.

"Carousel"

When we mention carousel, it means a series of transitioning actions such as pictures. These would usually evolve through stages of an animation which are generally staged in percentages.
So it would look like this:-
@keyframes carousel {
0% {background-image: url ('1st picture') }
25% {background-image: url ('2nd picture') }
50% {background-image: url ('3rd picture') }
75% {background-image: url ('4th picture') }
100% {background-image: url ('5th picture')
}

#carousel {
width:100px;
height:200px;
animation: carousel 50s infinite;
}

Cubic Beziers

Cubic beziers are the rate of which we can determine the speed transitions take place. They are generally equally spaced out which would be known as a "Linear cubic bezier". We can change the rates to suit our preferations.
The function we would use is the "animation-timing-function: cubic-bezier (.0,.0,.0,.0,)"
See cubic bezier to play around with the different modes.

Transitions

Starting with the "transition" property, we would need to specify 2 things. 1) The CSS property we would like to be effected. 2) The duration of the effect. This would have to have a value greater than 0.

With a transition effect applied, not only can we alter the colour features of an object, we can change many more properties along with it. For instance:-
If we were to use the code

.btn{
width: 100px;
height:25px;
background: green;
transition: background-color 3s, width 3s;
}


.btn:hover{
width:200px;
height:75px;
background: aqua;
}

In format with a form, you can see how it can make a difference to the user. Thus keeping them interacting with the page.







We could also use transitions in terms of enlarging text to ensure peoper choice on a page. By ahving enlarged text as the user hovers this can ensure that the link they are choosing is the desired one.

Transform

To elevate the life that we bring to our page we can even animate or "transform" items on our page.
Using the "Transform" property we can do many things such as:-
- Rotate = Rotate an item on the axis (X or Y). This would look like:-




- Translate = This would give a reflection if you will of the item dependent on the values given for the axis.






- Skew = Alters the viewing angle of an item. This usually only requires one value, either X (horizontal) or Y (vertical). This reflects using positive and negative values in degrees.





- Scale = The viewing size of an item that can increase or decrease dependent on the values used in the funtion. Positive values would increase the sizing and negative values would decrease the scaling.
